## 🎬 Flujo del Demo

1. **Introducción** (15s) - Mostrar el problema
2. **Demo en vivo** (30s) - Analizar solicitud
3. **Valor agregado** (30s) - Mostrar resultados
4. **Tecnología** (30s) - Explicar IBM Bob
5. **Cierre** (15s) - Call to action

- **Practica el timing** hasta que fluya naturalmente
- **Memoriza el pitch** de 30 segundos
